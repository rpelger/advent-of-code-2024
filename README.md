# Advent of Code 2024

These are my solution for the [Advent of Code 2024](https://adventofcode.com/2024)

All implementation is done in [Elixir](https://elixir-lang.org/)


## Run

To run a solved puzzle, do:

```
$ elixir day1/puzzle.exs
```
